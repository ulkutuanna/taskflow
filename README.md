# ✅ TaskFlow

> Java ile geliştirilmiş basit ve temiz bir Yapılacaklar Listesi uygulaması.

## 🚀 Özellikler
- 📋 Görevleri listele
- ➕ Yeni görev ekle
- 🗑️ Görev sil

## 🛠️ Kullanılan Teknoloji
- Java 17
- ArrayList, Scanner

## ⚙️ Nasıl Çalıştırılır?
1. Repoyu klonla: `git clone https://github.com/ulkutuanna/taskflow.git`
2. IntelliJ IDEA ile aç
3. `ToDoList.java` dosyasını çalıştır
